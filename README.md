# Ankit (Akki) — Developer Portfolio (Futuristic)

This is a starter futuristic developer portfolio built with React, Tailwind CSS, Framer Motion and Three.js (via @react-three/fiber and drei).

Features:
- Dark, neon-themed UI with glassmorphism
- Particle background and floating code snippets
- Typing effect for skills
- Animated cards and progress bars
- Simple 3D tech visualization

Get started (Windows cmd):

```cmd
cd c:\Users\Chatu\Portfolio\portfolio-site
npm install
npm run dev
```

Notes:
- Project links are placeholders. Replace GitHub/live links in `src/components/Projects.jsx`.
- The contact form is a demo — add a backend or email service to send messages.
